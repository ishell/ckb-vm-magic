# 课程地图：Rust + RISC-V 区块链 VM 工程训练

## 课程目标

1. 建立“共识约束优先”的架构直觉。
2. 掌握 `ckb-vm` 执行主线与关键边界。
3. 能独立完成一次优化改动的风险评审。

## 课程结构（建议 3 天）

### Day 1：执行与边界

- 主文第 1-4 章
- 图示长文批次一
- 实验手册批次一

### Day 2：系统与语义

- 主文第 5-8 章
- 图示长文批次二
- 实验手册批次二

### Day 3：治理与专题

- 主文第 9-11 章 + 附录 A/C
- 图示长文批次三
- 实验手册批次三
- 练习题与讲评

## 产出要求

- 每位学员提交：
  - 一份架构评审报告（1-2 页）
  - 一份实验记录表（最少 3 个实验）
  - 一份 PR 审计模板演练结果（附录 F）
