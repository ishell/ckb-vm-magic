# 教学版讲义与练习题

本目录用于教学和内部培训，面向“区块链 VM 工程师训练营”。

- `00-course-map.md`：课程地图与目标
- `01-lecture-notes.md`：授课讲义（模块化）
- `02-exercises.md`：练习题（分层）
- `03-answer-guide.md`：参考答案与评分建议
- `04-90min-instructor-script.md`：90 分钟授课脚本（逐分钟讲师动作）
- `05-90min-slide-outline.md`：90 分钟幻灯片大纲（含讲者提示）
- `06-90min-live-demo-runbook.md`：90 分钟现场演示 runbook（主路径 + 备用路径）
- `07-90min-assessment-sheet.md`：90 分钟课堂评估单（可直接打分）
- `08-90min-slide-verbatim-script.md`：90 分钟逐页口播稿（可直接照读）
