# 图示审计模板样例集

## 一针见血定位

模板解决的是“该检查什么”；样例解决的是“怎么把结论写到可签署”。

本目录给出三份完整样例：

1. 一份`有条件通过`案例（性能优化可上线但需补门禁）。
2. 一份`拒绝通过`案例（边界设计直接违背共识约束）。
3. 一份`真实锚点映射`案例（直接映射当前 ckb-vm 代码路径）。

## 使用方式

- 先选一份与当前 PR 最接近的样例。
- 再把其中“证据结构”和“决策逻辑”替换为你自己的数据。
- 禁止复制结论；只允许复用结构。
