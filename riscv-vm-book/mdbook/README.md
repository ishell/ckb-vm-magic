# mdBook 预览说明

## 内容结构

- 发布页：封面页、版本页、更新日志、审稿人摘要
- 主体章节：第 1-11 章（出版版，含深度扩展）
- 图示拆解长文：已完成批次一、批次二、批次三
- 图示审计模板：每篇图示长文对应一份审计模板（含通过/拒绝/真实锚点样例）
- 实验手册：通用准备 + 实验脚本化执行说明 + 批次一/二/三实验
- 教学资料：课程地图、讲义、练习题、参考答案、90 分钟教学包、逐页口播稿
- 附录 A：密码学专题
- 附录 F：密码学实现审计模板（PR 可直接使用）
- 附录 H：secp256k1 端到端案例
- 附录 I：哈希/Merkle 端到端案例
- 附录 C：RISC-V 专题
- 附录 B：术语总表（带锚点）
- 附录 D：术语交叉索引
- 附录 E：出版规范与合规矩阵
- 附录 G：作者终版改稿清单

## 依赖安装

如果本机尚未安装 `mdbook`，先执行：

```bash
cargo install mdbook
```

## 本地预览

```bash
cd docs/riscv-vm-book/mdbook
mdbook serve
```
