# Rust 与 RISC-V：区块链虚拟机的工程必然性

本书基于 `ckb-vm` 源码，系统分析区块链 VM 的操作系统化设计，并回答两个核心问题：

1. 为什么共识执行层倾向选择 Rust？
2. 为什么 RISC-V 能成为可治理的长期 ISA 方案？
